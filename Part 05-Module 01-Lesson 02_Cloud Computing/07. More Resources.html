You can use your AWS Promotional Credits to get started with AWS Machine Learning services such as Amazon SageMaker. Amazon SageMaker is a fully-managed machine learning platform that enables developers and data scientists to quickly and easily build, train, and deploy machine learning models at any scale. It removes all the barriers that typically slow down developers who want to use machine learning, so it's a perfect starting point on your machine learning journey.
